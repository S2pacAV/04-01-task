# Tf-IDF

$$W_{x, D} = \mathrm{tf}_{x, D} \times \log  \dfrac{N}{\mathrm{df}_{x}}$$

* $x$ ‚Äì —Å–ª–æ–≤–æ
* $D$ ‚Äì –¥–æ–∫—É–º–µ–Ω—Ç
* $N$ ‚Äì –æ–±—â–µ–µ —á–∏—Å–ª–æ –¥–æ–∫—É–º–µ–Ω—Ç–æ–≤
* $\mathrm{df}_{x}$ ‚Äì —á–∏—Å–ª–æ –¥–æ–∫—É–º–µ–Ω—Ç–æ–≤, —Å–æ–¥–µ—Ä–∂–∞—â–∏—Ö $x$
