# –ö–≤–∞–Ω—Ç–∏–ª—å —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è(ppf)

_–î–ª—è –ì–°_:

**–ú–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–µ –æ–∂–∏–¥–∞–Ω–∏–µ**(—Å—Ä–µ–¥–Ω–µ–µ): $\mathbb{E}[X], \mathbb{M}[X]$

* –î–ª—è –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ–π —Å–ª—É—á–∞–π–Ω–æ–π –≤–µ–ª–∏—á–∏–Ω—ã $\mathbb{E}[X] = \int_{-\infty}^{+\infty}x_i f(x_i)$

_–î–ª—è –≤—ã–±–æ—Ä–∫–∏_:

–í—ã–±–æ—Ä–æ—á–Ω–æ–µ —Å—Ä–µ–¥–Ω–µ–µ —è–≤–ª—è–µ—Ç—Å—è –Ω–∞–∏–ª—É—á—à–µ–π –æ—Ü–µ–Ω–∫–æ–π –º–∞—Ç.–æ–∂–∏–¥–∞–Ω–∏—è –ì–°, –∏–Ω–∞—á–µ –≥–æ–≤–æ—Ä—è –ª—É—á—à–µ –≤—Å–µ–≥–æ –ø—Ä–∏–±–ª–∏–∂–∞–µ—Ç —ç—Ç–æ –∑–Ω–∞—á–µ–Ω–∏–µ (**np.mean()**)

**–í—ã–±–æ—Ä–æ—á–Ω–æ–µ —Å—Ä–µ–¥–Ω–µ–µ**: $\bar{X} = \dfrac{1}{n}\sum_{i=1}^{n}x_i$

> üî•`np.mean()`

# –û—Å–Ω–æ–≤–Ω—ã–µ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∏ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–π

**–î–∏—Å–ø–µ—Ä—Å–∏—è**: $\mathbb{D}[X] = \mathbb{E}[(X - \mathbb{E}[X])^2] = \mathbb{E}[X^2] - (\mathbb{E}[X])^2$

* –ú–µ—Ä–∞ _—Ä–∞–∑–±—Ä–æ—Å–∞_ —Å–ª—É—á–∞–π–Ω–æ–π –≤–µ–ª–∏—á–∏–Ω—ã
* –ß–∞—Å—Ç–æ –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è –∫–∞–∫ $\mathbb{D}[X]$, $Var [X]$ (variance)

_–î–ª—è –≤—ã–±–æ—Ä–∫–∏_:

–í—ã–±–æ—Ä–æ—á–Ω–∞—è –¥–∏—Å–ø–µ—Ä—Å–∏—è —è–≤–ª—è–µ—Ç—Å—è –Ω–∞–∏–ª—É—á—à–µ–π –¥–∏—Å–ø–µ—Ä—Å–∏–∏ –ì–°, –∏–Ω–∞—á–µ –≥–æ–≤–æ—Ä—è –ª—É—á—à–µ –≤—Å–µ–≥–æ –ø—Ä–∏–±–ª–∏–∂–∞–µ—Ç —ç—Ç–æ –∑–Ω–∞—á–µ–Ω–∏–µ (**np.var()**)
**–í—ã–±–æ—Ä–æ—á–Ω–∞—è –¥–∏—Å–ø–µ—Ä—Å–∏—è**: $\mathbb{s}^2 = \dfrac{1}{n-1}\sum_{i=1}^{n}(x_i - \bar{X})^2$

> üî•`np.var(ddof=1)` (–ø—Ä–∏ ddof=1 –≤ –∑–Ω–∞–º–µ–Ω–∞—Ç–µ–ª–µ —Ñ–æ—Ä–º—É–ª—ã n-1)

**–°—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–µ –æ—Ç–∫–ª–æ–Ω–µ–Ω–∏–µ**: $\sigma_x = \sqrt{\mathbb{D}[X]}$

* –≤—ã—á–∏—Å–ª—è–µ—Ç—è –ø—Ä–æ—Å—Ç–æ –∫–∞–∫ –∫–æ—Ä–µ–Ω—å –∏–∑ –¥–∏—Å–ø–µ—Ä—Å–∏–∏

_–î–ª—è –≤—ã–±–æ—Ä–∫–∏_:
* –∞–Ω–∞–ª–æ–≥–∏—á–Ω–æ –∏–∑–≤–ª–µ—á–µ—Ç—å –∫–æ—Ä–µ–Ω—å –∏–∑ –≤—ã–±–æ—Ä–æ—á–Ω–æ–π –¥–∏—Å–ø–µ—Ä—Å–∏–∏

> üî•`np.std(ddof=1)` (–ø—Ä–∏ ddof=1 –≤ –∑–Ω–∞–º–µ–Ω–∞—Ç–µ–ª–µ —Ñ–æ—Ä–º—É–ª—ã n-1)
